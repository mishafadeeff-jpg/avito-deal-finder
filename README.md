# Avito Deal Finder AI - ИИ-Мониторинг выгодных объявлений Авито

Проект предназначен для поиска недооцененных товаров (iPhone, Авто, Электроника) и моментального оповещения перекупов в Telegram.

## Структура проекта:
- `config.py` - настройки бота, токен Telegram, пороги скидок
- `database.py` - хранение базы проверенных объявлений и подписок
- `avito_scanner.py` - парсер и ИИ-аналитик цен
- `bot.py` - главная логика Telegram-бота
